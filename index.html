What is folding@home?
folding@home is a distributed computing project that simulates protein dynamics including folding. Researchers use the results from the project to find treatments for diseases, including COVID-19.
Technology
Distributed computing projects use the processing power on users' PCs to work through demanding workloads. The combined power of every folding@home donor is greater than the 7 most powerful supercomputers combined, with 470 petaflops.
Biology
From their website, folding@home "is a distributed computing project for simulating protein dynamics, including the process of protein folding and the movements of proteins implicated in a variety of diseases.
Download
Windows: Install from their website just as a normal application. You can launch the control app or go to client.foldingathome.org to view process and set preferences. This website has some issues with chrome. It works fine on firefox, and, interesting, Microsoft Edge.
Arch Based Linux Distros: folding@home is in the AUR. Install with pamac install foldingathome. Run it from the command line as root. To set it up, run sudo ./FAHClient --configure. Make sure to install CUDA (Nvidia GPU) or OpenCL (AMD GPU) to use the GPU. You can control it from client.foldingathome.org.
Ubuntu Based Linux Distros: Download and install the .deb's from their website and run sudo ./FAHClient --configure to set up the app.
MacOS: Go to their website, download, install. There may configuration steps that I'm unaware of because I've never set this up on MacOS.
